## Quick Start

### Experiment 3.1.1
```bash
cd experiment-3.1.1/login-form
npm install && npm start
```
Open http://localhost:3000 — login with `admin@test.com` / `password123`

---

### Experiment 3.1.2
```bash
# Terminal 1 – Backend
cd experiment-3.1.2/jwt-auth-app/server
npm install && node server.js

# Terminal 2 – Frontend
cd experiment-3.1.2/jwt-auth-app/client
npm install && npm start
```
Open http://localhost:3000 — login with `admin` / `password123`

Note: Run both backend and frontend terminals for this experiment.

---

### Experiment 3.1.3
```bash
# Terminal 1 – Backend
cd experiment-3.1.3/rbac-app/server
npm install && node server.js

# Terminal 2 – Frontend
cd experiment-3.1.3/rbac-app/client
npm install && npm start
```

Note: Run both backend and frontend terminals for this experiment.
